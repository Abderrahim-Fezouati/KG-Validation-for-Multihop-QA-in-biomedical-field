# Graph-CoRAG (Clean Starter)

## Structure
- src/graphcorag/
  - text_retriever.py
  - dense_retriever.py
- scripts/
  - run_hybrid.py
- config/
  - umls_dict.txt
  - umls_dict.overlay.json
  - relation_schema.json
- data/
  - corpus.jsonl
  - kg_edges.merged.csv
  - queries.autogenerated.20.jsonl (optional, dev)

## Run (example)
python -u scripts/run_hybrid.py 
  --proj "F:\new experiment with Kg2c dataset" 
  --corpus "data/corpus.jsonl" 
  --kg "data/kg_edges.merged.csv" 
  --dict "config/umls_dict.txt" 
  --overlay "config/umls_dict.overlay.json" 
  --schema "config/relation_schema.json" 
  --queries "data/queries.autogenerated.20.jsonl" 
  --out "out/run_autogen20" 
  --topk 80 
  --min_constraints 2 
  --mode both 
  --bm25_mod_path "src/graphcorag/text_retriever.py" 
  --dense_mod_path "src/graphcorag/dense_retriever.py"
